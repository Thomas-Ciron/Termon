## Executable options
Usage : `termon [OPTION]...`
Play Pokemon on terminal.

With no OPTION, play Pokemon.
- `-l` or `--language` : set the language
- `-h` or `--help` : display help and exit
- `-v` or `--version` : display version information and exit
